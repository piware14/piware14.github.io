Pi-Ware

Pi-Ware is an app store for the Raspberry Pi that allows you to install apps that you can't get using sudo apt-get install in Terminal.

Looking for the Pi-Ware applist?

Pi-Ware

Installing:

To install Pi-Ware, run the command below:

git clone https://github.com/piware14/pi-ware.git
cd pi-ware
bash install

Updating:

Pi-Ware will automatically update.

To force update Pi-Ware, run the command below:

bash $HOME/pi-ware/updater force

Uninstalling:

To uninstall Pi-Ware, run the command below:

pi-ware --uninstall-all
